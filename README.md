# 인터넷 서점 <북적북적> 사이트
 본 프로젝트는 오라클을 데이터베이스로 이용, JSP와 서블릿으로 구현한 인터넷 서점 사이트 제작 프로젝트입니다.

---

### 참여

김주혁(조장) 김다은 이현우 오수경 조아라 최진영 

---

### 구현 목표 기능

#### 사용자

1. 기본적인 회원기능
   1. 로그인
   2. 로그아웃
   3. 회원가입
2. 검색기능
3. 장바구니 기능
4. 종이책 주문/환불/취소/배송조회 기능
5. 바로드림 주문/환불/취소 기능
6. E북 주문/환불/취소/읽기 기능
7. 마이페이지 기능
   1. 포인트내역 조회
   2. 설정한 관심분야 조회
8. 중고거래게시판 기능
   1. 중고거래글 등록/수정/삭제 기능
   2. 중고거래글의 댓글 등록/수정/삭제 기능
9. 고객센터
   1. 자주하는 질문 게시판 조회 기능
   2. QnA 게시판 글 작성/수정/삭제 기능
   3. 채팅상담 기능
10. 추천도서
   1. 설정한 관심분야에 따라 도서를 추천하는 기능

#### 관리자

1. 도서 정보 관리 기능
   1. 도서 추가/수정/삭제 기능
2. e북 정보 관리 기능
   1. e북 추가/수정/삭제 기능
3. 주문 관리 기능
   1. 주문 내역 조회 기능
   2. 주문 상태 변경 기능
4. 회원 정보 관리 기능
   1. 회원 상태 변경 기능
5. 통계 기능
6. 공지사항 게시판 관리 기능
   1. 공지사항 게시판 글 작성/수정/삭제 기능
7. 중고거래게시판 관리 기능
   1. 중고거래게시판 글 및 댓글 삭제 기능
8. 자주하는 질문 게시판 관리 기능
   1. 자주하는 질문 글 작성/수정/삭제 기능
   2. 질문카테고리 추가/수정 기능
9. 채팅상담 기능
10. 큐레이션 레터(이메일 전송 서비스) 기능

---

### 개발환경

운영체제 : Windows10, MacOS   Catalina 10.15.7  

개발툴 : Eclipse(OJDBC6)

개발언어 : JAVA(JDK 1.8)

데이터베이스 : Oracle

서버 : Apache Tomcat 8.5
